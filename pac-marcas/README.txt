Cómo cargar el sitio:

No necesitas ningún servidor web para probarlo localmente. Simplemente haz doble clic en el archivo index.html, o ábrelo con tu navegador favorito.

Requisitos:

Ninguno. Todo lo necesario está incluido en los archivos del proyecto. Solo necesitas un navegador moderno (como Chrome, Firefox, Safari o Edge).
